üîß Troubleshooting

Printer Not Detected?

  • Check USB connection and printer power
  • Restart the app: Right-click tray icon ‚Üí Restart
  • Manual refresh: Settings ‚Üí "Refresh Printers"

Connection Failed?

  • Check firewall settings - allow the app through
  • Verify internet connection to 31.57.166.18:3001
  • Re-validate license: Settings ‚Üí "Re-validate License"

Build Failed?

Windows Users: Install Visual Studio Build Tools if USB driver compilation fails
Mac Users: Install Xcode Command Line Tools: xcode-select --install
Linux Users: Install build essentials: sudo apt-get install build-essential
